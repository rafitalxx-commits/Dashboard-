# Dashboard - Proyecto

Repositorio oficial del dashboard. Incluye módulo de tareas en prototipo independiente lista para integrar.

## Módulos
- `dashboard/` — panel principal
- `modules/tasks/` — módulo de tareas
- `docs/` — documentación técnica

## Inicio rápido
```bash
npm install
npm run dev
```

## Despliegue
Ver `docs/deploy.md`

## Ramas
- `main` — producción
- `dev` — integración
- `feature/*` — funcionalidades
